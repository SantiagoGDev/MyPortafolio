Conocimientos

Actualmente mis cononocimientos en el desarrollo web son basicos, la mayoria de mis conocimientos vienen siendo autodidactas con practicas que me propongo para ir mejorando mis habiliadades como desarrollador.

HTML

80%

HyperText Markup Language, es un lenguaje de marcado creado por Tim Berners-Lee, con el cual se le da a estructura a las páginas Web a traves de etiquetas.

CSS

60%

Cascading Style Sheet, es un lenguaje estilos creado por Håkon Wium Lie y Bert Bos, que tiene como proposito dar vida a las estructuras creadas en HTML.

JavaScript

30%

Es un lenguaje de programación que vive en el navegador creado por Brendan Eich, es un lneguaje importante en el desarrollo web ya que es el que se encarga de dar funcionalidades o dinamismo a las páginas web.

Git

60%

Es un sistema de control de versiones Open Source el cual fe creado por Linus Torvalds el cual al finalizar la creacion de Linux decidio crear este sistema para tener mas control sobre los proyectos de una manera sencilla.
